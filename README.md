# 江门第三方报表 - 车辆运行监控情况

## 项目简介

车辆运行监控情况的报表展示系统，用于展示车辆运行监控数据。系统提供了车队选择、日期筛选、数据搜索、分页展示等功能，并支持数据导出。

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
- Bootstrap Icons

## 项目结构

```
报表项目/
├── index.html          # 主页面
├── css/
│   └── style.css       # 自定义样式
├── js/
│   ├── api.js          # API接口封装
│   └── main.js         # 主要业务逻辑
├── API.txt             # API文档
└── README.md           # 项目说明
```

## 功能特点

1. **车队选择**：支持选择不同车队查看数据
2. **日期筛选**：支持选择日期范围查看数据
3. **数据展示**：以表格形式展示车辆运行监控数据
4. **违规标识**：违规记录特殊标识和突出显示
5. **分页功能**：支持分页查看数据，可调整每页显示数量
6. **数据统计**：显示总记录数、违规车辆数、正常车辆数等统计信息
7. **数据导出**：支持导出数据为Excel文件


## 使用说明

1. 打开 `index.html` 文件即可访问系统
2. 选择车队和日期范围，点击"搜索"按钮查看数据
3. 使用快速搜索框可以按关键词搜索数据
4. 点击表格列标题可以进行排序
5. 违规记录会以红色背景标识，并显示"违规"标签
6. 使用分页控件可以切换页面，或调整每页显示数量
7. 点击"导出"按钮可以导出当前筛选条件下的数据

## API对接

项目已经封装了与后端API对接的接口，位于 `js/api.js` 文件中。在实际部署时，需要：

1. 修改 `js/api.js` 中的 `BASE_URL` 为实际的API地址
2. 根据实际API返回的数据结构调整 `js/main.js` 中的 `processApiData` 函数
3. 确保API接口与前端请求参数和返回数据格式匹配

## 主要API接口

1. `getOrganizations()` - 获取组织列表
2. `getVehicleMonitoringDetails(params)` - 获取车辆运行监控明细
3. `exportVehicleMonitoringDetails(params)` - 导出车辆运行监控明细
4. 其他相关API接口详见 `js/api.js` 文件
